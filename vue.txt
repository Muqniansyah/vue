🔍 Apa itu Vue.js?

Vue.js adalah framework JavaScript untuk membangun antarmuka web yang interaktif dan responsif. Ringan, mudah dipelajari, dan cocok untuk pemula.

Vue adalah kerangka kerja JavaScript front-end yang ditulis dalam JavaScript.
Kerangka kerja yang mirip dengan Vue adalah React dan Angular, tetapi Vue lebih ringan dan lebih mudah untuk memulai.

🌟 Kenapa Vue?

Mudah dipelajari (dibanding Angular atau React)

Dokumentasi lengkap dan bahasa manusia

Komunitas aktif

Bisa digunakan dari proyek kecil sampai besar


PEMBAHASAN :
1. installasi, via cdn atau nodejs.
2. interpolasi teks (menampilkan pesan)
3. vue directives (terdiri dari v-bind, v-if, v-show, v-for, v-on, v-model)
4. v-bind
5. v-if
