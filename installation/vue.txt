3.  Setelah proyek dibuat, ikuti petunjuk untuk menginstal dependensi dan memulai server pengembangan :
$ cd <your-project-name>    (masuk ke folder)
$ npm install               (melakukan installasi package)
$ npm run dev               (menjalankan website)

4. Saat Anda siap mengirimkan aplikasi Anda ke produksi, jalankan yang berikut ini:
$ npm run build

Ini akan membuat versi aplikasi Anda yang siap produksi di direktori proyek ./dist.
