Contoh: 'v-for' untuk membuat li-tag

Dalam contoh ini, 'v-for' membuat tag 'li' dengan nama makanan untuk setiap item makanan dalam array instance Vue.

  1. {{ makanan }}

penjelasan :

makanan in banyakMakanan artinya: untuk setiap elemen (makanan) yang ada di dalam array banyakMakanan, buatkan 'li'.

makanan adalah nama variabel sementara untuk mewakili tiap item dalam array saat loop berjalan. Bisa kamu ganti namanya sesuka hati. misalnya:

v-for="food in banyakMakanan">{{ food }}

Hasilnya sama saja, cuma lebih jelas karena namanya sesuai konteks.

pada {{ makanan }}, yaitu Sintaks {{ ... }} disebut interpolasi di Vue. Itu digunakan untuk menampilkan nilai dari variabel ke dalam HTML. Jadi di setiap loop, {{ makanan }} akan menampilkan elemen array pada posisi tersebut.
