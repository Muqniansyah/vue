Contoh: Dapatkan indeks elemen array dengan 'v-for'

Nomor indeks elemen array bisa sangat berguna dalam for-loop JavaScript. Untungnya, kita v-for juga bisa mendapatkan nomor indeks saat menggunakannya di Vue. Untuk mendapatkan nomor indeks, v-for kita perlu memberikan dua kata yang dipisahkan koma dalam tanda kurung: kata pertama akan menjadi elemen array, dan kata kedua akan menjadi indeks elemen array tersebut.

Direktif 'v-for' digunakan untuk mendapatkan indeks dan nama makanan dari elemen di dalam array 'manyFoods' pada instance Vue.

{{ index }}: "{{ x }}"


Kita juga dapat menampilkan indeks elemen array dan informasi dari elemen array itu sendiri, jika elemen array tersebut adalah objek

Direktif 'v-for' digunakan untuk mendapatkan indeks objek di dalam array 'manyFoods2', bersama dengan nama dan url setiap objek makanan.

{{ index }}: "{{ x.name }}", url: "{{ x.url }}"
