Contoh dengan JavaScript di dalam {{ }}

Teks 'message' dari instance Vue digabungkan dengan teks lain dan ekspresi JavaScript. Ekspresi JavaScript ini menghasilkan bilangan bulat acak antara 1 dan 6. Ekspresi JavaScript sederhana juga dapat ditulis di dalam kurung kurawal ganda {{ }}.

{{ message }}
{{'Random number: ' + Math.ceil(Math.random()*6) }}
